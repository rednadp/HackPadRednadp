# HackPadRednadp
I have tried to create a pad similar to the stremdeck, or at least that was de idea. I have never made something similar.
# Bill of Materials
 - Seeed XIAO RP2040
 - Through-hole 1N4148 Diodes (x9)
 - MX-Style switches (x9)
 - EC11 Rotary encoder
 - 0.91 inch OLED display
 - Blank DSA keycaps
 - SK6812 MINI-E LEDs (x8)
 - M3x16mm screws (x4)
 - M3x5mx4mm heatset inserts (x4)
 - Printed parts
